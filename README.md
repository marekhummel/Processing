# Processing
Collection of all the small Processing projects I've got. I've migrated all sketches to use the python library `py5`. This way we can run this in almost native processing syntax, but without the application (just python and the venv).

It includes a folder called Coding challenges, which represents a collection of solutions to the challenges given by Daniel Shiffman in his YT playlist (https://www.youtube.com/playlist?list=PLRqwX-V7Uu6ZiZxtDDRCi6uhfTH4FilpH)
